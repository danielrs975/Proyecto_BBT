Como jugar?

Para jugar se necesita correr el siguiente comando 

        ruby main.rb

A partir de aqui se mostrara un menu en la terminal en donde puede elegir
    -   Si jugar con la computadora
    -   O, Simular un juego PC VS PC

Si selecciona el primero se le pedira que introduzca una estrategia para la 
PC, luego se le pedira que modalidad de partida se jugara si por rondas o limite
de puntos. Por cada una de las rondas se le pedira a usted que ingrese una jugada
que son:
    -   Piedra
    -   Papel
    -   Tijera
    -   Spock 
    -   Lagarto 

Escribala tal cual se muestra aqui para que el juego pueda ocurrir sin problemas

Para simular un juego PC VS PC se le pedira que introduzca dos estrategias y lo que 
necesite las estrategias que haya escogido. Luego la modalidad de la partida y se le
mostrara los resultados al final del juego
